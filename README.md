# Samarth Portfolio

This is a clean Next.js project initialized for **Samarth's Portfolio**.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js
- **Styles**: Tailwind CSS
- **Font**: Geist
